# Maintainers

This repository is a community-maintained fork of `aequasi/terraform-provider-discord` via `Chaotic-Logic/terraform-provider-discord`.

## Current Maintainers

* @45ck

## Maintainer Responsibilities

* Triage issues, review PRs, and keep CI green.
* Keep docs and import ID formats accurate.
* Be explicit about Terraform semantics (drift, delete, write-only fields).
* Cut releases and publish to the Terraform Registry namespace (when applicable).

## Adding / Removing Maintainers

Maintainers may be added or removed by consensus of the current maintainers.
